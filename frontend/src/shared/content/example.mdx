# 欢迎使用 MDX

这是一个 **MDX** 示例文件。MDX 让你可以在 Markdown 中使用 React 组件！

## 什么是 MDX？

MDX 是 Markdown 的超集，它允许你：

- 在 Markdown 中导入和使用 React 组件
- 导出变量和组件
- 使用 JSX 语法

## 代码示例

这是一段内联代码：`const hello = "world"`

这是一个代码块：

```tsx
function HelloWorld() {
  return <h1>Hello, MDX!</h1>;
}
```
