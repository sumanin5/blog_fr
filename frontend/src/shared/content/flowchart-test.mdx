# 流程图测试页面

这是一个简单的测试页面，用于验证流程图功能是否正常工作。

## Mermaid 流程图测试

### 基本流程图

```mermaid
graph TD
    A[开始] --> B{是否登录?}
    B -->|是| C[显示主页]
    B -->|否| D[跳转登录页]
    D --> E[用户输入账号密码]
    E --> F{验证成功?}
    F -->|是| C
    F -->|否| G[显示错误信息]
    G --> E
    C --> H[结束]
```

### 时序图测试

```mermaid
sequenceDiagram
    participant U as 用户
    participant F as 前端
    participant B as 后端

    U->>F: 点击登录
    F->>B: 发送请求
    B-->>F: 返回结果
    F-->>U: 显示页面
```
