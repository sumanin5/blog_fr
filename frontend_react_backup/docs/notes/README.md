## 🎯 如何使用这些笔记

### 遇到问题时

1. 先查看相关笔记，看是否有类似问题
2. 按照笔记中的解决方案尝试修复
3. 如果问题仍未解决，查看"调试技巧"部分

### 学习新知识时

1. 阅读笔记中的"原理详解"部分
2. 查看"实际例子"加深理解
3. 参考"最佳实践"应用到项目中

### 添加新笔记

当你遇到新问题或学到新知识时：

1. 在 `docs/notes/` 目录下创建新的 `.md` 文件
2. 使用清晰的标题和结构
3. 包含：问题描述、原理解释、解决方案、例子
4. 更新本 README 的笔记列表

## 🤝 贡献指南
